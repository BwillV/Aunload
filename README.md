# AUnload Rescue

App de inventario para warehouse con captura de códigos AUnload.

## Descargar la app

👉 **[Descargar última versión](https://github.com/BwillV/Aunload/releases/latest)**

## Características

- ✅ Captura automática de códigos AUnload desde el escáner Urovo DT50X
- ✅ Reconocimiento automático del número de container desde Receiving/Put Away
- ✅ Sincronización en tiempo real vía Firebase para trabajo en equipo
- ✅ Sistema de auto-actualización integrado
- ✅ Botón flotante para acceso rápido
- ✅ Widget para la pantalla de inicio

## Instalación

1. Descarga el APK desde [Releases](https://github.com/BwillV/Aunload/releases/latest)
2. Instala en tu dispositivo Android
3. Activa el Servicio de Accesibilidad en Configuración
4. Ingresa el código de equipo en Ajustes para sincronización

## Actualización

La app verifica automáticamente actualizaciones al abrirse. Solo toca "Actualizar" cuando aparezca el diálogo.
